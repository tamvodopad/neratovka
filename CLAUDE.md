# neratovka.ru

Сайт села Нератовка (Заинский район, Татарстан). Этнографический проект Кости Попова.

## Что есть

- `html/index.html` — placeholder корня (noindex, пока пусто)
- `html/ww2.html` — Книга Памяти, 110 уроженцев Нератовки в ВОВ. Самостоятельный HTML, ~80 КБ, без сборки

## Архитектура

Статика. Никакой сборки, никакого JS-фреймворка. Один HTML — один экран. Шрифты с Google Fonts (PT Serif + Old Standard TT). Данные встроены как `<script type="application/json">`.

## Эстетика ww2.html

- «Архивный мемориал» — состаренная бумага, красный знамённый, золото медалей
- Никаких ярких цветов или анимаций кроме сдержанных reveal-on-scroll
- Имена + типографика — основной носитель смысла. Иллюстраций нет

## Данные

Источник: «Память народа» (pamyat-naroda.ru). Изначально 126 архивных записей, после ручной дедупликации — 110 уроженцев. Логика дублей: одинаковое имя+отчество+близкий год → один человек, ссылки сохраняются как `extra_links` в записи.

При добавлении новых уроженцев:
1. Найти запись на pamyat-naroda.ru
2. Добавить объект в `<script id="data">` с полями `n, y, b, s, t, l, aw, el`
3. Обновить числа в hero / dedication / stat-cards вручную
